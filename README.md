# BBS BATTLE CHAOS

匿名掲示板に「投資」を足したらどうなるか、を試してる個人開発のゲームです。

普通の掲示板みたいに誰でも匿名でスレを立てて書き込めます。違うのは、登録した人が「これは面白い」と思ったレスにゲーム内のお金を投資して、株を買えるところ。後から投資が集まったレスほど株価が上がるので、まだ誰も気づいてない名レスを早めに仕込んだ人が得をします。要するに目利き勝負です。

あと、スレもレスも放っておくと時間で朽ちて消えます。盛り上がってるうちに動かないと、買った株も紙くずになる。そういう焦りも込みで遊ぶやつです。

> 念のため：ゲーム内のお金や株に現実の価値はありません。換金もできないし、賭博でもないです。

## ざっくりルール

- スレ立て・書き込みは匿名・無料。ログインも要りません。
- 投資（＝株購入）は要ログイン。初期資金が配られます。
- 株価は累計投資額で上がる（早く買うほど安い）。
- レスもスレもHPがあって、時間とともに減る。0になると消滅。
- 投資が集まったレスはレベルが上がって寿命が延びる。
- 相場には「天候」があって、荒れてると朽ちるのが速い。
- 人が少ないうちは運営のNPCが投資して場を動かす（ランキングには入らない）。

## 技術構成

- PHP 8 / MySQL 8 / nginx、全部 Docker
- フレームワークは使わず、ルーター・ビュー・DIあたりは自前。DIコンテナだけ PHP-DI
- クリーンアーキテクチャ寄りの構成（Domain / Application / Infrastructure / Presentation）
- テストは PHPUnit

なんで自作かというと、勉強がてら中身を全部見えるようにしたかったからです。
Claude Codeで作ってもらいつつ、脆弱性は自分がチェックしている感じです。

## ローカルで動かす

Docker があれば動きます。

```bash
cp .env.example .env   # 中身を環境に合わせて埋める
docker compose up -d
```

`migrate` サービスがDBのスキーマを流してから `php` が立ち上がります。起動したらブラウザで:

```
https://localhost:18443
```

開発用の証明書は自己署名なので、ブラウザに警告が出ます。承認して進んでください。HTTP(18723)はHTTPSへ飛ばしてます。

メールはローカルだと Mailpit で受け取れます（確認メール・パスワード再設定など）。UIは `http://localhost:18025`。

## よく使うコマンド

```bash
# テスト
composer test
# または
docker compose exec php vendor/bin/phpunit

# マイグレーション（手動で流したいとき）
docker compose exec php php bin/migrate.php

# ラウンドのリセット（終局してたら確定→初期化→再開）
docker compose exec php php bin/round.php          # 終局時のみ
docker compose exec php php bin/round.php --force  # 強制
```

`bin/round.php` は投稿・スレ・株を消すので、流す先のDBを間違えないこと。

## 設定（.env）

`.env.example` にひととおりコメント付きで書いてあります。要点だけ:

- `DB_PASSWORD` などは本番だと弱いと起動を蹴られます。
- メールは本番だと実送信(SMTP)必須。
- Googleログイン、ゲームバランス、法務ページの運営者名などは環境変数で切り替え。詳しくは下のドキュメント。

## ディレクトリ

```
src/
  Domain/          ルール・エンティティ（外部に依存しない核）
  Application/     ユースケース・ポート
  Infrastructure/  DB・メール・OAuth・ログなどの実装
  Presentation/    ルーティング・コントローラ・ビュー・多言語
bin/               マイグレーション / ラウンド用のCLI
database/migrations/  連番SQL
tests/             ユニット中心。結合テストはDB必須（普段はスキップ）
docs/              設計と運用メモ
```

日本語・英語の両対応です（ヘッダーから切り替え）。
