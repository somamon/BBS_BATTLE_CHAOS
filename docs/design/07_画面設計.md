# 画面設計

> 本書のコア仕様はすべて [20_リデザイン_朽ちると投資.md](20_リデザイン_朽ちると投資.md) に従う（特に §2 主要メカニクス / §6 画面・ルート）。
> ゲームは **2層構造**（匿名掲示板層 ＋ 投資家層）。スレ・レスは個別 HP で朽ちて消滅し、登録ユーザーはスレの株を売買して総資産を競う。
> 世界フェーズ（boom/calm/storm/crash）が相場天候として減衰倍率に効き、変異（化け株 Lv0〜3）でスレが強化される。
> 数値・名称・パスは doc20 と完全一致させる（[§3 バランス定数](20_リデザイン_朽ちると投資.md) 参照）。

---

## 1. 画面一覧

| 画面 | パス | メソッド | 認証 | 概要 |
|---|---|---|---|---|
| スレ一覧 | `/threads` | GET | 匿名可 | alive スレを HPバー・勢い・時価総額・変異バッジ付きで一覧（dead 除外） |
| スレ詳細 | `/thread/{id}` | GET | 匿名可 | スレHP/変異/総株数＋レス一覧＋レス投稿フォーム＋投資パネル |
| スレ作成 | `/threads`（フォームは一覧/専用） | POST | 匿名可 | タイトルを入力してスレ作成（HP満タン, total_shares=0） |
| レス投稿 | `/thread/{id}/posts` | POST | 匿名可 | スレ詳細内フォームから投稿（無料・無報酬） |
| 投資 | `/thread/{id}/invest` | POST | 要登録 | スレ詳細の投資パネルから amount を投資 |
| 登録 | `/register` | GET/POST | — | email + name + password で登録（所持金 500 付与） |
| ログイン | `/login` | GET/POST | — | email + password でログイン |
| マイページ | `/me` | GET | 要登録 | 所持金・保有株・評価額・含み損益・総資産 |
| ランキング | `/ranking` | GET | 匿名可 | 全ユーザーの総資産（所持金＋株評価額）降順 |
| リザルト | `/result` | GET | 匿名可 | 終局メッセージ＋最終ランキング＋勝者 |

> POST 系（スレ作成・レス投稿・投資・登録・ログイン）は **CSRF トークン必須**（doc20 §6）。

## 2. 共通レイアウト（ヘッダー）

全画面共通のヘッダーに、**現在の相場フェーズ**と、ユーザーの所持金（または登録導線）を常時表示する。

- **相場フェーズ表示**: `boom` / `calm` / `storm` / `crash` を常時バッジ表示。フェーズで全スレの減衰スピードが変わる旨を示す（boom=朽ちにくい〜crash=暴落）。
- **未登録時**: `[ 登録して投資 ]` ボタンを表示（→ `/register`）。
- **登録済み時**: 所持金（`money`）を常時表示し、`/me`・`/ranking`・ログアウト導線を出す。

```
┌──────────────────────────────────────────────────────────┐
│  朽ちる掲示板        相場: [ STORM ×1.8 ⚡急速に朽ちる ]     │
│  ───────────────────────────────────────────────────────  │
│  未登録 → [ 登録して投資 ]   /   登録済 → 所持金 ¥420  ☰    │
└──────────────────────────────────────────────────────────┘
```

| フェーズ | 倍率 | ヘッダー体感表記 |
|---|---|---|
| `boom` | ×0.7 | 活況・朽ちにくい（強気相場） |
| `calm` | ×1.0 | 平穏・通常 |
| `storm` | ×1.8 | 嵐・急速に朽ちる（延命コスト増） |
| `crash` | ×2.5 | 暴落・投げ売り（投資妙味は最大） |

> フェーズはアクセス時の遅延遷移で更新される（doc20 §2.5）。ヘッダーは「いまの相場」を映すだけ。

## 3. スレ一覧（`/threads`）

### 要素

- alive スレのみ表示（**dead は除外・復活不可**）。
- 各スレ: タイトル、**HPバー**（現在HP/max_hp）、**残り寿命の目安**、**勢い**（レス数＝`post_count`）、**時価総額**（＝現在HP）、**変異バッジ**（Lv1〜3）。
- 現在HPは遅延減衰でその場計算（フェーズ倍率込み）。`storm`/`crash` ではバーの減りが速い旨を注記。
- スレ作成導線。

### ワイヤーフレーム

```
┌──────────────────────────────────────────────────────────┐
│ ヘッダー（相場: CRASH ×2.5 / 所持金 ¥420）                  │
├──────────────────────────────────────────────────────────┤
│  スレッド一覧            ※相場 CRASH 中はHPの減りが最速     │
│                                            [ + スレを立てる ]│
│  ┌──────────────────────────────────────────────────────┐ │
│  │ 雑談スレ                            🧬 変異Lv2 特異点前 │ │
│  │ HP ████████░░ 3200/4000   時価総額 3200               │ │
│  │ 勢い 156レス   残り寿命 ~約11分（相場CRASH反映）        │ │
│  ├──────────────────────────────────────────────────────┤ │
│  │ 質問スレ                                   （変異なし）  │ │
│  │ HP ██████████ 980/1000    時価総額 980                 │ │
│  │ 勢い 3レス     残り寿命 ~約8分                          │ │
│  ├──────────────────────────────────────────────────────┤ │
│  │ ニュース速報スレ                        🧬 変異Lv1 変異種│ │
│  │ HP ███░░░░░░░ 540/2000    時価総額 540                 │ │
│  │ 勢い 88レス    残り寿命 ~約4分（朽ち接近・投げ売り注意） │ │
│  └──────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────┘
```

> 残り寿命の目安 ≒ `現在HP ÷ (decay_per_min × phase_multiplier)` 分。表示は概算で可（doc20 §2.1）。

## 4. スレ詳細（`/thread/{id}`）

### 要素

- **スレ状態**: 現在HP/max_hp、**変異Lv**（0〜3）、**総株数**（`total_shares`）、status。
- **レス一覧**: 各レスに HP（個別に朽ちて消える）、投稿者識別、本文。dead レスは表示しない。
- **レス投稿フォーム**: 本文入力＋送信（匿名可・無料）。スレが dead なら投稿不可。
- **投資パネル（要登録）**: `amount` 入力 → **配分プレビュー**（HP回復50% / 配当30% / sink20%）、自分の持ち分株と評価額、投資ボタン。未登録時は `[ 登録して投資 ]` を表示。

### ワイヤーフレーム

```
┌──────────────────────────────────────────────────────────┐
│ ヘッダー（相場: STORM ×1.8 / 所持金 ¥420）         ← 一覧へ │
├──────────────────────────────────────────────────────────┤
│  雑談スレ                                                  │
│  ┌─ スレ状態 ───────────────────────────────────────────┐ │
│  │ HP ████████░░ 3200/4000   変異Lv2（タイトル発光）       │ │
│  │ 総株数 2480   status: alive                            │ │
│  └──────────────────────────────────────────────────────┘ │
│                                                            │
│  ┌─ レス一覧（流れて消える）─────────────────────────────┐ │
│  │ 1: aB3x9k   HP ███████░░░ 71/100                      │ │
│  │    こんにちは、このスレは伸びてますね                  │ │
│  │ 2: kF7m2p   HP ████░░░░░░ 38/100                      │ │
│  │    今のうちに仕込むべきか…                             │ │
│  │ 3: 山田(登録)HP █░░░░░░░░░ 12/100  ←まもなく消滅       │ │
│  │    変異Lv3が見えてきた                                 │ │
│  └──────────────────────────────────────────────────────┘ │
│  ┌─ レス投稿 ───────────────────────────────────────────┐ │
│  │ [_______________________________________]  [ 投稿する ]│ │
│  └──────────────────────────────────────────────────────┘ │
│                                                            │
│  ┌─ 投資パネル（要登録）────────────────────────────────┐ │
│  │ 投資額 amount: [ 100 ]                                 │ │
│  │ 配分プレビュー:                                        │ │
│  │   HP回復 50% →  +50   （max_hp 上限・超過分はsinkへ）  │ │
│  │   配当   30% →   30   （既存株主へ持ち分比率で分配）   │ │
│  │   sink   20% →   20   （経済から消滅）                 │ │
│  │ 受取株: +100株   投資後の持ち分 280/2580株             │ │
│  │ あなたの評価額: 約 347（持ち分比率 × 現在HP）          │ │
│  │                                        [ 投資する ]    │ │
│  └──────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────┘
```

> 評価額 = `(my_shares ÷ total_shares) × 現在HP`（doc20 §2.3）。最初の投資者は既存株主が居ないため、その回の配当30%は sink に合流（doc20 §2.2）。

## 5. 登録（`/register`）／ ログイン（`/login`）

### 要素

- 登録: email・表示名(name)・password。成功で所持金 500 付与＋セッション紐付け。失敗＝メール重複/形式不正。
- ログイン: email・password。`password_verify()` 一致でセッションに user_id 保存。失敗＝不在/不一致。

### ワイヤーフレーム

```
┌────────────────────────────┐   ┌────────────────────────────┐
│  新規登録                   │   │  ログイン                   │
│  email   [______________]  │   │  email   [______________]  │
│  表示名  [______________]  │   │  password[______________]  │
│  password[______________]  │   │            [ ログイン ]     │
│  ※登録すると所持金500付与   │   │  → 未登録なら [ 新規登録 ]  │
│            [ 登録する ]     │   │                            │
└────────────────────────────┘   └────────────────────────────┘
```

## 6. マイページ（`/me`、要登録）

### 要素

- **所持金**（money）。
- **保有株テーブル**: スレ / 持ち分株（my_shares / total_shares）/ **評価額** / **含み損益**。
- **総資産** = 所持金 ＋ Σ株評価額。

### ワイヤーフレーム

```
┌──────────────────────────────────────────────────────────┐
│ ヘッダー（相場: CALM ×1.0 / 所持金 ¥420）                  │
├──────────────────────────────────────────────────────────┤
│  マイページ（山田）                                        │
│  所持金: ¥420                                              │
│  ┌─ 保有株 ─────────────────────────────────────────────┐ │
│  │ スレ          持ち分株      評価額    含み損益          │ │
│  │ 雑談スレ      280/2580      347       +112  ▲          │ │
│  │ 質問スレ       50/ 800       61       − 9   ▽          │ │
│  │ 速報スレ(dead) 0株（紙くず）  0       −100  ✕          │ │
│  └──────────────────────────────────────────────────────┘ │
│  総資産: ¥420 + 株408 = 828                                │
└──────────────────────────────────────────────────────────┘
```

> dead スレの株は評価額0（紙くず）。含み損益は累計投資との対比で示す目安表示。

## 8. リザルト（`/result`）

終局条件（全スレ dead、または 所持金合計 < MIN_INVEST）を満たすとラウンドが確定し、本画面に遷移する（doc20 §8）。

### 要素

- **終局メッセージ**（ラウンド終了の告知）。
- **最終ランキング**（総資産降順）。
- **勝者**（総資産トップ）を大きく表示。
- 以降は新ラウンド開始まで閲覧のみ。

### ワイヤーフレーム


## 10. HPバー・変異の見せ方（指針）

### HPバー（色）

| 現在HP割合（現在HP/max_hp） | 色 | 体感 |
|---|---|---|
| 60%以上 | 緑 | 健全・優良株 |
| 30〜60% | 黄 | 減衰進行・注視 |
| 10〜30% | 橙 | 朽ち接近・投げ売り警戒 |
| 10%未満 | 赤（点滅） | まもなく dead |

- バーは現在HPを遅延計算で描画。`storm`/`crash` 中は「減りが速い」旨をスレ近傍に注記し、相場とHP挙動を結び付けて見せる。
- レスの HP も同様にバー表示し、低HPは「まもなく消滅」を明示（流れて消える体験を可視化）。

### 変異（化け株）の演出

doc20 §2.6 の演出列に対応させ、Lv が上がるほど視覚的に強化する。

| 変異Lv | バッジ／演出 | 補足 |
|---|---|---|
| 0 | 通常表示 | バッジなし |
| 1 | 「変異種」バッジ | max_hp 2000・配当ボーナス ×1.1 |
| 2 | タイトル発光（グロー） | max_hp 4000・配当ボーナス ×1.2 |
| 3 | 「特異点」化（強い発光＋枠装飾） | max_hp 8000・配当ボーナス ×1.3 |

- 変異判定は投資トランザクション内で発火するため、投資直後に Lv が上がった場合は詳細画面でバッジ昇格を強調して見せる。
- 一覧では変異バッジで優良株を一目で識別できるようにし、「変異目前」の総株数も勢いとして読めるようにする。
