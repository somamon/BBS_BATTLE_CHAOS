### コンポーネント

| コンポーネント | 更新方式 | 間隔 |
|---|---|---|
| スレ状態（HP/汚染） | HTMX polling | 3秒 |
| イベント通知 | 投稿レスポンス | 即時 |
| 投稿一覧 | HTMX polling | 3秒 |
| 世界状態（小） | HTMX polling | 3秒 |

### 投稿フォームの挙動

1. 投稿ボタン押下 → `POST /api/post`（HTMX）
2. レスポンスで投稿一覧の末尾に追加
3. イベント発生時はイベント通知エリアにフラッシュ表示（3秒で消える）
4. フォームはリセット

## UI演出設計

### 汚染度による演出

| 汚染度 | 演出 |
|---|---|
| 0〜20 | 通常表示 |
| 21〜40 | テキストが微かに揺れる（CSS animation） |
| 41〜60 | 背景にノイズ（CSS filter） |
| 61〜80 | 色反転エリアがランダム出現 |
| 81〜100 | 文字崩壊、グリッチエフェクト全開 |

### 天候による演出

| 天候 | 演出 |
|---|---|
| sunny | 通常（明るい背景） |
| red_rain | 背景に赤い粒子、テキスト色が赤寄り |
| fog | 半透明のオーバーレイ、視認性低下 |
| storm | 画面が微振動、稲光エフェクト |
| void | 背景が真っ黒、テキストが浮かぶ |
| blood_moon | 全体が暗い赤、影が濃い |
| static | TV砂嵐のノイズ、テキストがちらつく |

### フェーズによる演出

| フェーズ | 演出 |
|---|---|
| calm | 通常UI |
| unstable | ヘッダーが微かに揺れる |
| chaos | UI全体が歪む、色がランダムに変化 |
| apocalypse | 画面全体が激しく揺れ、亀裂エフェクト |

### 深夜帯演出（0:00〜5:00）

| 演出 | 説明 |
|---|---|
| 背景暗転 | 全体的に暗くなる |
| UI点滅 | ランダムにUIパーツが点滅 |
| 怪文書表示 | 画面端に意味不明なテキストがフラッシュ |
| 環境音 | （将来）不気味なアンビエント音 |

### イベント発生時演出

| イベント | 演出 |
|---|---|
| meteor | 画面上から赤い光が落下するアニメーション |
| blackout | 画面が一瞬真っ暗になる |
| earthquake | 画面全体が揺れる（CSS transform） |
| garbled_text | 対象テキストがグリッチする |
| healing_light | 画面が一瞬白く光る |

## CSS設計方針

| 方針 | 説明 |
|---|---|
| BEMライク命名 | `.block__element--modifier` |
| CSS変数活用 | 天候・フェーズでCSS変数を切り替え |
| アニメーション | `@keyframes` で定義、Alpine.js でトリガー |
| レスポンシブ | モバイルファースト、単カラム |

### CSS変数例

```
:root {
  --bg-color: #1a1a2e;
  --text-color: #e0e0e0;
  --accent-color: #0f3460;
  --danger-color: #e94560;
  --corruption-color: #6b0f1a;
}

/* 天候: 赤い雨 */
[data-weather="red_rain"] {
  --bg-color: #2a0a0a;
  --text-color: #ff6666;
  --accent-color: #8b0000;
}

/* フェーズ: chaos */
[data-phase="chaos"] {
  --bg-color: #0a0a0a;
  animation: phase-chaos 2s infinite;
}
```

## Alpine.js 使用箇所

| 用途 | 説明 |
|---|---|
| イベントフラッシュ | 表示→自動消去のタイマー管理 |
| 投稿フォーム | バリデーション表示、送信状態管理 |
| 演出制御 | 天候・汚染度に応じたクラス切り替え |
| タイマー | お題の残り時間カウントダウン |
