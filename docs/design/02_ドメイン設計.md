# ドメイン設計

## Entity 一覧

### Thread（スレッド）

スレッドは「生物」。HPを持ち、状態変化し、死ぬ。

**プロパティ**

| プロパティ | 型 | 説明 | 初期値 |
|---|---|---|---|
| id | ThreadId (VO) | 一意識別子 | ULID自動生成 |
| title | ThreadTitle (VO) | タイトル | — |
| hp | HitPoint (VO) | 体力 | 100 |
| corruption | Corruption (VO) | 汚染度 | 0 |
| mutationLevel | MutationLevel (VO) | 変異レベル | 0 |
| weather | Weather (VO) | スレ固有天候 | "晴れ" |
| status | ThreadStatus (Enum) | 状態 | active |
| createdAt | DateTimeImmutable | 作成日時 | — |

**振る舞い**

| メソッド | 説明 |
|---|---|
| takeDamage(int $amount) | HPを減らす。0以下で崩壊状態に遷移 |
| heal(int $amount) | HPを回復。最大100 |
| addCorruption(int $amount) | 汚染度を加算。最大100 |
| mutate() | 変異レベルを1上げる。最大5で mutated 状態に |
| changeWeather(Weather $weather) | 天候を変更 |
| collapse() | 崩壊状態に遷移 |
| freeze() | 凍結状態に遷移 |
| isAlive(): bool | active状態かどうか |
| canPost(): bool | 投稿可能か（active かつ frozen でない） |

**ドメインルール**

- HP は 0〜100 の範囲
- 汚染度は 0〜100 の範囲
- 変異レベルは 0〜5 の範囲
- collapsed / frozen 状態のスレには投稿不可
- 汚染度 80 以上でイベント発生率 2倍

### Post（投稿）

投稿は「行動」。世界に影響を与えるトリガー。

**プロパティ**

| プロパティ | 型 | 説明 |
|---|---|---|
| id | PostId (VO) | 一意識別子 |
| threadId | ThreadId (VO) | 所属スレッドID |
| content | PostContent (VO) | 本文 |
| authorHash | AuthorHash (VO) | 投稿者識別ハッシュ |
| isCorrupted | bool | 汚染済みフラグ |
| createdAt | DateTimeImmutable | 投稿日時 |

**振る舞い**

| メソッド | 説明 |
|---|---|
| corrupt() | 投稿内容を汚染状態にする |
| containsKeyword(string $keyword): bool | 特定キーワードを含むか判定 |

**ドメインルール**

- 本文は1〜1000文字
- 空投稿不可
- 汚染済み投稿は表示時に文字化け処理が入る

### WorldState（世界状態）

掲示板全体が持つ唯一の世界状態。シングルトン的存在。

**プロパティ**

| プロパティ | 型 | 説明 | 初期値 |
|---|---|---|---|
| weather | Weather (VO) | 世界天候 | "晴れ" |
| dangerLevel | DangerLevel (VO) | 危険度 | 0 |
| anomalyLevel | AnomalyLevel (VO) | 異常度 | 0 |
| phase | WorldPhase (Enum) | フェーズ | calm |
| updatedAt | DateTimeImmutable | 最終更新 | — |

**振る舞い**

| メソッド | 説明 |
|---|---|
| increaseDanger(int $amount) | 危険度を加算 |
| decreaseDanger(int $amount) | 危険度を減算 |
| increaseAnomaly(int $amount) | 異常度を加算 |
| changeWeather(Weather $weather) | 天候変更 |
| updatePhase() | 危険度に基づきフェーズを自動遷移 |
| isNightTime(): bool | 現在時刻が深夜帯（0:00〜5:00）か |
| getEventRateMultiplier(): float | 現在の状態に基づくイベント発生率倍率 |

**フェーズ自動遷移ルール**

| 危険度 | フェーズ |
|---|---|
| 0〜29 | calm |
| 30〜59 | unstable |
| 60〜89 | chaos |
| 90〜100 | apocalypse |

## ドメインサービス

### EventJudgeService

投稿時のイベント発生を判定する。

**責務**

- 乱数に基づくイベント抽選
- 世界状態・スレ状態に応じた発生率調整
- 深夜帯の発生率ブースト
- 特定キーワードによるイベントトリガー

**判定ロジック概要**

```
基本発生率: 15%
  × 世界危険度倍率（1.0〜3.0）
  × 汚染度倍率（1.0〜2.0）
  × 深夜帯倍率（1.0 or 1.5）
  × フェーズ倍率（calm:1.0 / unstable:1.2 / chaos:1.5 / apocalypse:2.0）

結果 > 乱数(0〜100) → イベント発生
```

### ChallengeJudgeService

お題ルールへの適合を判定する。

**責務**

- 投稿内容のルール適合チェック
- 違反時のペナルティ種別決定
- 複数お題同時発動時の判定

### WorldPhaseService

世界状態のフェーズ遷移を管理する。

**責務**

- 危険度に基づくフェーズ自動遷移
- フェーズ遷移時のイベント発火
- apocalypse 完了後のリセット処理
