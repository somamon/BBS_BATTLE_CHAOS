## コンセプト

**「朽ちる掲示板 × 投資経済（株式モデル）」**。

匿名掲示板の賑わいそのものを「将来性シグナル」とみなし、その上に株式投資ゲームを乗せた **2層構造**の作品。

```
┌─ 投資家層（登録ユーザーのみ）────────────────┐
│  お金(初期 500)で「スレッドの株」を売買し、配当と  │
│  株評価益で総資産を競う。                          │
└──────────────────▲─────────────────┘
                       │ 賑わいが「将来性シグナル」になる
┌─ 匿名掲示板層（誰でも）──┴─────────────────┐
│  匿名でスレ立て・レス投稿（お金不要・報酬なし）。     │
│  スレもレスも時間で朽ちて消える。                   │
└──────────────────────────────────┘
```

### 3つの概念

1. **朽ちる** — スレもレスも個別に HP を持ち、時間で減り、0 で消滅する。減衰は cron ではなく「読むたびに経過時間から計算する」遅延減衰（Lazy Decay）。
2. **投資** — 登録ユーザーがスレッドの株を買う。投資額は **HP回復 50% / 既存株主への配当 30% / 消滅(sink) 20%** に分かれ、投資者は `amount` 株を取得する。株評価額 ＝ 持ち分比率 × スレの現在HP。
3. **お金** — 登録時に 500。レス・スレ作成は無報酬。**自分が株主のスレに新規投資が入ると配当**で増える。

### 競争（ゴール）

レス報酬を 0、配当を再分配（新規マネーを生まない）にしたため、お金は構造的にデフレ＝**有限ラウンド**。お金が枯れる／全スレが朽ちると終局し、**総資産（所持金 ＋ 保有株の評価額）トップが勝者**となる。

## コアの相乗要素

旧「異常掲示板」のカオス要素は、世界を彩る2つの**相場メカニクス**として再解釈・採用した。

- **世界フェーズ（相場天候）** — 世界に1行だけの相場環境（`boom` / `calm` / `storm` / `crash`）が、全スレの減衰に倍率（0.7〜2.5）を掛ける。嵐・暴落は延命コストを跳ね上げ、投資家を一斉に動かす。
- **変異（化け株）** — スレの累計投資（`total_shares`）が閾値（500 / 2000 / 8000）を超えると変異し（`mutation_level` 0〜3）、`max_hp` が上昇し配当ボーナスが付く。「変異目前のスレを先読みして仕込む＝化け株を当てる」興奮。

> 汚染・お題・連鎖イベントなどの旧カオス要素は、コアを汚さない「相場イベント層」として将来拡張に回す（詳細は [20_リデザイン_朽ちると投資.md](20_リデザイン_朽ちると投資.md) §10）。

## ターゲット / 狙い

**投機の興奮を匿名掲示板に乗せる。** スレが伸びるうちに乗って、朽ちる前に配当を回収する——「伸びる板を当てる」スリルと、相場天候・化け株が生む読み合いを、誰でも気軽に書ける匿名掲示板の賑わいの上で味わわせる。賑わいが資産になり、放置は減衰（損）になる、**有限ラウンドの資産運用ゲーム**として遊んでもらうことを狙う。
