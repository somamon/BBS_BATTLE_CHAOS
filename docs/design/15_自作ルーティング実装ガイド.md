## 1. 前提知識：PHPスーパーグローバル

スーパーグローバルは「どのスコープからでも参照できる組み込み配列」。ルーティングで使うのは主に以下。

### 1.1 `$_SERVER` — リクエスト/サーバ情報

ルーターで実際に使うキーだけ厳選。

| キー | 例 | 用途 |
|---|---|---|
| `$_SERVER['REQUEST_METHOD']` | `"GET"` / `"POST"` | HTTPメソッド判定 |
| `$_SERVER['REQUEST_URI']` | `"/thread/42?page=2"` | **クエリ込み**のパス。`parse_url` でパス部だけ取り出す |
| `$_SERVER['QUERY_STRING']` | `"page=2"` | `?` 以降の文字列（通常 `$_GET` を使うので直接は使わない） |
| `$_SERVER['REMOTE_ADDR']` | `"172.18.0.1"` | クライアントIP（プロキシ手前の素のIP） |
| `$_SERVER['HTTP_X_FORWARDED_FOR']` | `"203.0.113.5"` | プロキシ経由の元IP（**信頼できる時だけ**使う） |
| `$_SERVER['HTTP_*']` | `$_SERVER['HTTP_HX_REQUEST']` | 任意のリクエストヘッダ。`ヘッダ名` を **大文字＋ハイフンをアンダースコア** にして `HTTP_` を前置 |
| `$_SERVER['CONTENT_TYPE']` | `"application/json"` | ボディの形式判定 |
| `$_SERVER['HTTPS']` | `"on"` or 未設定 | https かどうか |

> **ヘッダ名の変換ルールが重要**：`HX-Request` ヘッダ → `$_SERVER['HTTP_HX_REQUEST']`、`Content-Type` → `$_SERVER['CONTENT_TYPE']`（Content-Type と Content-Length だけは例外的に `HTTP_` が付かない）。

### 1.2 `$_GET` / `$_POST` / `php://input`

| 取得元 | 中身 | 使う場面 |
|---|---|---|
| `$_GET` | URLの `?key=value` をパースした連想配列 | クエリパラメータ |
| `$_POST` | `application/x-www-form-urlencoded` / `multipart/form-data` のフォーム値 | HTMLフォーム送信 |
| `php://input` | リクエストボディの生データ（ストリーム） | JSON（`application/json`）を受ける時。`file_get_contents('php://input')` で読み、`json_decode` する |

```php
// JSON ボディを受ける場合
$raw  = file_get_contents('php://input');     // 生ボディ
$json = json_decode($raw, true) ?? [];        // 連想配列に（失敗時は空配列）
```

### 1.3 その他

| 変数 | 用途 |
|---|---|
| `$_COOKIE` | クッキー値 |
| `$_SESSION` | セッション。`session_start()` 後に使える。CSRFトークン・レートリミットの保存先 |
| `$_FILES` | アップロードファイル（今回は未使用想定） |

## 2. 前提知識：使用する関数・構文

このルーターで登場するものを全部解説する。

### 2.1 関数

| 関数 | 役割 | 例 |
|---|---|---|
| `parse_url($url, PHP_URL_PATH)` | URLからパス部分だけ抜く | `parse_url('/a/b?x=1', PHP_URL_PATH)` → `'/a/b'` |
| `rawurldecode($s)` | `%E3%82%B9` 等を元の文字に戻す | 日本語URL対応 |
| `rtrim($s, '/')` | 末尾の `/` を削る | `rtrim('/thread/', '/')` → `'/thread'` |
| `explode($sep, $s)` | 文字列を区切りで配列化 | `explode('/', '/a/b')` → `['','a','b']` |
| `preg_match($re, $s, $m)` | 正規表現マッチ。`$m` にキャプチャが入る。返り値 1=一致 | 後述 |
| `preg_replace_callback($re, $cb, $s)` | マッチ部分をコールバックの戻り値で置換 | パス→正規表現変換に使用 |
| `preg_quote($s, '#')` | 正規表現の特殊文字をエスケープ | `.` `/` 等をリテラル化 |
| `http_response_code($n)` | HTTPステータスコード設定 | `http_response_code(404)` |
| `header($s)` | レスポンスヘッダ送信（**出力前に呼ぶ**） | `header('Content-Type: text/html')` |
| `array_reduce($arr, $cb, $init)` | 配列を1つの値に畳み込む | ミドルウェア合成に使用 |
| `hash_equals($a, $b)` | タイミング攻撃に強い文字列比較 | CSRF検証 |
| `random_bytes($n)` / `bin2hex()` | 安全な乱数 → 16進文字列 | CSRFトークン生成 |

### 2.2 構文（PHP 8 系）

```php
// 配列の分割代入（destructuring）
[$controller, $action] = $route->handler;

// null合体演算子：左がnull/未定義なら右
$method = $_SERVER['REQUEST_METHOD'] ?? 'GET';

// null合体代入
$_SESSION['rate'] ??= [];

// match式（switchの式版・厳密比較）
$status = match (true) {
    $code < 300 => 'ok',
    $code < 400 => 'redirect',
    default     => 'error',
};

// アロー関数（1式・外部変数を自動キャプチャ）
$fn = fn($x) => $x * 2;

// 無名関数（use で外部変数を取り込む）
$next = function ($req) use ($handler) { return $handler($req); };

// 名前付き引数
new Response(status: 404, body: 'Not Found');

// 可変長引数 / スプレッド
function add(...$args) {}
$obj->method(...$params);

// readonly プロパティ（コンストラクタ昇格）
public function __construct(public readonly string $method) {}

// 第一級callable（PHP8.1）
$ref = $controller->index(...);
```

### 2.3 正規表現の名前付きキャプチャ（ルーティングの核心）

```php
$path = '/thread/42';
if (preg_match('#^/thread/(?P<id>[^/]+)$#', $path, $m)) {
    echo $m['id'];   // "42"  ← (?P<id>...) の "id" で取り出せる
}
```

| 記号 | 意味 |
|---|---|
| `#...#` | 区切り文字（`/` を含むパスを扱うので `/` 以外を区切りに使う） |
| `^` `$` | 行頭・行末（＝完全一致を強制） |
| `(?P<name>...)` | 名前付きキャプチャグループ |
| `[^/]+` | スラッシュ以外1文字以上（1セグメント分） |
| `\d+` | 数字1文字以上（型制約用） |

## 3. Request クラス

`$_SERVER` 等の生データを全部ここに閉じ込める。以降アプリ本体は `$_GET` 等を直接触らない。

`src/Presentation/Http/Request.php`

```php
<?php

declare(strict_types=1);

namespace App\Presentation\Http;

final class Request
{
    /** @var array<string,string> resolve() でルーターが注入するパスパラメータ */
    private array $params = [];

    public function __construct(
        private readonly string $method,
        private readonly string $path,
        private readonly array $query,   // $_GET
        private readonly array $body,    // $_POST or JSON
        private readonly array $server,  // $_SERVER
        private readonly array $cookies, // $_COOKIE
    ) {}

    /** スーパーグローバルから生成するファクトリ */
    public static function fromGlobals(): self
    {
        // REQUEST_URI はクエリ込みなのでパス部分だけ取り出し、URLデコードする
        $uri  = $_SERVER['REQUEST_URI'] ?? '/';
        $path = rawurldecode(parse_url($uri, PHP_URL_PATH) ?: '/');

        // JSONボディならパースして body にする
        $contentType = $_SERVER['CONTENT_TYPE'] ?? '';
        if (str_contains($contentType, 'application/json')) {
            $body = json_decode(file_get_contents('php://input') ?: '', true) ?? [];
        } else {
            $body = $_POST;
        }

        return new self(
            method:  strtoupper($_SERVER['REQUEST_METHOD'] ?? 'GET'),
            path:    $path,
            query:   $_GET,
            body:    $body,
            server:  $_SERVER,
            cookies: $_COOKIE,
        );
    }

    public function method(): string { return $this->method; }
    public function path(): string   { return $this->path; }

    /** GETパラメータ（?key=value） */
    public function query(string $key, mixed $default = null): mixed
    {
        return $this->query[$key] ?? $default;
    }

    /** POST/JSONボディの値 */
    public function input(string $key, mixed $default = null): mixed
    {
        return $this->body[$key] ?? $default;
    }

    /** パスパラメータ（/thread/{id} の id） */
    public function param(string $key, mixed $default = null): mixed
    {
        return $this->params[$key] ?? $default;
    }

    /** ルーターがマッチ後に呼ぶ：パスパラメータを差し込んだ新インスタンスを返す */
    public function withParams(array $params): self
    {
        $clone = clone $this;
        $clone->params = $params;
        return $clone;
    }

    /** ヘッダ取得：'HX-Request' → $_SERVER['HTTP_HX_REQUEST'] へ変換して引く */
    public function header(string $name): ?string
    {
        $key = 'HTTP_' . strtoupper(str_replace('-', '_', $name));
        return $this->server[$key] ?? null;
    }

    /** クライアントIP（基本は REMOTE_ADDR） */
    public function ip(): string
    {
        return $this->server['REMOTE_ADDR'] ?? '0.0.0.0';
    }

    /** HTMXリクエストか（HX-Request ヘッダの有無） */
    public function isHtmx(): bool
    {
        return $this->header('HX-Request') !== null;
    }
}
```

## 4. Response クラス

ステータス・ヘッダ・ボディを溜め込み、最後に `send()` で1回だけ出力する。

`src/Presentation/Http/Response.php`

```php
<?php

declare(strict_types=1);

namespace App\Presentation\Http;

final class Response
{
    /** @var array<string,string> */
    private array $headers = [];

    public function __construct(
        private string $body = '',
        private int $status = 200,
    ) {}

    public static function html(string $body, int $status = 200): self
    {
        $res = new self($body, $status);
        $res->headers['Content-Type'] = 'text/html; charset=UTF-8';
        return $res;
    }

    public static function json(array $data, int $status = 200): self
    {
        $res = new self(json_encode($data, JSON_UNESCAPED_UNICODE), $status);
        $res->headers['Content-Type'] = 'application/json; charset=UTF-8';
        return $res;
    }

    public static function redirect(string $url, int $status = 302): self
    {
        $res = new self('', $status);
        $res->headers['Location'] = $url;
        return $res;
    }

    public static function error(int $status, string $message): self
    {
        return self::html("<h1>{$status}</h1><p>{$message}</p>", $status);
    }

    public function withHeader(string $name, string $value): self
    {
        $this->headers[$name] = $value;
        return $this;
    }

    /** 実際にクライアントへ送る（最後に1回だけ呼ぶ） */
    public function send(): void
    {
        http_response_code($this->status);
        foreach ($this->headers as $name => $value) {
            header("{$name}: {$value}");
        }
        echo $this->body;
    }
}
```

## 6. Route / RouteMatch

`src/Presentation/Routing/Route.php`

```php
<?php

declare(strict_types=1);

namespace App\Presentation\Routing;

final class Route
{
    public readonly string $regex;
    /** @var string[] */
    public readonly array $paramNames;
    public readonly bool $isStatic;

    /**
     * @param array{0:class-string,1:string} $handler [Controller::class, 'action']
     * @param string[] $middlewares
     */
    public function __construct(
        public readonly string $method,
        public readonly string $path,
        public readonly array $handler,
        public readonly array $middlewares = [],
    ) {
        [$this->regex, $this->paramNames, $this->isStatic] = self::compile($path);
    }

    /**
     * パスパターンを正規表現へコンパイル。
     * "/thread/{id}" → "#^/thread/(?P<id>[^/]+)$#"
     * @return array{0:string,1:string[],2:bool}
     */
    private static function compile(string $path): array
    {
        $paramNames = [];

        // セグメントごとに処理：固定部は preg_quote、{name} は名前付きキャプチャに変換
        $segments = explode('/', $path);
        $converted = array_map(function (string $seg) use (&$paramNames) {
            // {name} または {name:正規表現}
            if (preg_match('#^\{(\w+)(?::(.+))?\}$#', $seg, $m)) {
                $paramNames[] = $m[1];
                $pattern = $m[2] ?? '[^/]+';
                return '(?P<' . $m[1] . '>' . $pattern . ')';
            }
            return preg_quote($seg, '#');
        }, $segments);

        $regex = '#^' . implode('/', $converted) . '$#';
        $isStatic = $paramNames === [];

        return [$regex, $paramNames, $isStatic];
    }
}
```

## 7. Router 本体

中核。登録・解決（resolve）・実行（dispatch）を持つ。

`src/Presentation/Routing/Router.php`

```php
<?php

declare(strict_types=1);

namespace App\Presentation\Routing;

use App\Presentation\Http\Request;
use App\Presentation\Http\Response;

final class Router
{
    /** @var array<string, array<string, Route>> 静的ルート [method][path] => Route */
    private array $static = [];
    /** @var array<int, Route> 動的ルート（登録順） */
    private array $dynamic = [];

    /**
     * @param (callable(class-string):object)|null $resolver
     *        コントローラ生成方法。null なら new。php-di を使うなら fn($c) => $container->get($c)
     */
    public function __construct(
        private readonly mixed $resolver = null,
    ) {}

    public function add(string $method, string $path, array $handler, array $mw = []): void
    {
        $method = strtoupper($method);
        $path   = $this->normalize($path);
        $route  = new Route($method, $path, $handler, $mw);

        if ($route->isStatic) {
            $this->static[$method][$path] = $route;
        } else {
            $this->dynamic[] = $route;
        }
    }

    public function get(string $path, array $handler, array $mw = []): void
    {
        $this->add('GET', $path, $handler, $mw);
    }

    public function post(string $path, array $handler, array $mw = []): void
    {
        $this->add('POST', $path, $handler, $mw);
    }

    /** 末尾スラッシュ除去（"/" だけは残す） */
    private function normalize(string $path): string
    {
        $path = rtrim($path, '/');
        return $path === '' ? '/' : $path;
    }

    /**
     * パス＋メソッドからルートを解決。
     * @throws NotFoundException どのパスにも一致しない
     * @throws MethodNotAllowedException パスは一致するがメソッド不一致
     */
    public function resolve(string $method, string $path): RouteMatch
    {
        $method = strtoupper($method);
        $path   = $this->normalize($path);

        // 1) 静的ルート：O(1) で先引き（静的が常に動的より優先される）
        if (isset($this->static[$method][$path])) {
            return new RouteMatch($this->static[$method][$path], []);
        }

        // 2) 動的ルート：登録順に正規表現マッチ
        $allowed = [];   // パスは一致したが method 違いのものを集める（405用）
        foreach ($this->dynamic as $route) {
            if (preg_match($route->regex, $path, $m)) {
                if ($route->method === $method) {
                    // 名前付きキャプチャだけ取り出す（数値キーは捨てる）
                    $params = [];
                    foreach ($route->paramNames as $name) {
                        $params[$name] = $m[$name];
                    }
                    return new RouteMatch($route, $params);
                }
                $allowed[] = $route->method;
            }
        }

        // 3) 静的側でもパス一致・メソッド違いを拾う（405の取りこぼし防止）
        foreach ($this->static as $m2 => $routes) {
            if (isset($routes[$path])) {
                $allowed[] = $m2;
            }
        }

        if ($allowed !== []) {
            throw new MethodNotAllowedException(array_values(array_unique($allowed)));
        }
        throw new NotFoundException();
    }

    /** 解決 → ミドルウェア → コントローラ実行 → Response */
    public function dispatch(Request $request): Response
    {
        $match   = $this->resolve($request->method(), $request->path());
        $request = $request->withParams($match->params);

        // 最内：コントローラ呼び出し
        $core = function (Request $req) use ($match): Response {
            [$class, $action] = $match->route->handler;
            $controller = $this->resolver ? ($this->resolver)($class) : new $class();
            $result = $controller->$action($req);
            // 文字列が返ってきたら 200 HTML に正規化
            return $result instanceof Response ? $result : Response::html((string) $result);
        };

        // ミドルウェアを外→内に合成（array_reduce で内側から包む）
        $pipeline = array_reduce(
            array_reverse($match->route->middlewares),
            function (callable $next, string $mwName): callable {
                $middleware = MiddlewareFactory::make($mwName);   // 名前 → インスタンス
                return fn(Request $req) => $middleware->handle($req, $next);
            },
            $core
        );

        return $pipeline($request);
    }
}
```

> **resolve の肝**：
> - 静的を `isset` で先に引く＝「`/thread/create` が `/thread/{id}` より必ず優先」になり、登録順問題が起きない。
> - 「パス一致・メソッド違い」を `$allowed` に集めてから 404/405 を判定するのが 405 を正しく返すコツ。

## 8. Middleware

`src/Presentation/Routing/Middleware/MiddlewareInterface.php`

```php
<?php

declare(strict_types=1);

namespace App\Presentation\Routing\Middleware;

use App\Presentation\Http\Request;
use App\Presentation\Http\Response;

interface MiddlewareInterface
{
    /** 通すなら $next($request) を返す。止めるなら自前の Response を返す。 */
    public function handle(Request $request, callable $next): Response;
}
```

例：CSRF ミドルウェア

```php
<?php

declare(strict_types=1);

namespace App\Presentation\Routing\Middleware;

use App\Presentation\Http\Request;
use App\Presentation\Http\Response;

final class CsrfMiddleware implements MiddlewareInterface
{
    public function handle(Request $request, callable $next): Response
    {
        // GET等は検証不要。POST系のみ検証
        if ($request->method() === 'POST') {
            $sent    = (string) $request->input('_csrf', '');
            $session = (string) ($_SESSION['_csrf'] ?? '');
            // hash_equals: タイミング攻撃に強い比較
            if ($session === '' || !hash_equals($session, $sent)) {
                return Response::error(403, '不正なリクエストです');
            }
        }
        return $next($request);   // 通過
    }
}
```

ミドルウェア名 → インスタンスの単純なファクトリ（DI導入前のつなぎ）：

```php
<?php

declare(strict_types=1);

namespace App\Presentation\Routing;

use App\Presentation\Routing\Middleware\CsrfMiddleware;
use App\Presentation\Routing\Middleware\MiddlewareInterface;

final class MiddlewareFactory
{
    public static function make(string $name): MiddlewareInterface
    {
        // "rate_limit:post" のように ":" 引数を含む場合は分解
        [$key, $arg] = array_pad(explode(':', $name, 2), 2, null);

        return match ($key) {
            'csrf'       => new CsrfMiddleware(),
            // 'rate_limit' => new RateLimitMiddleware($arg),  // 実装時に追加
            default      => throw new \InvalidArgumentException("unknown middleware: {$name}"),
        };
    }
}
```

コントローラ側の新しい書き方（`Request` を受け `Response` を返す）：

```php
<?php

namespace App\Presentation\Controller;

use App\Presentation\Http\Request;
use App\Presentation\Http\Response;

class ThreadController
{
    public function show(Request $request): Response
    {
        $id = $request->param('id');                 // /thread/{id} の id
        return Response::json(['thread_id' => $id]);
    }
}
```

## 10. index.php（全部つなぐ）

`public/index.php`

```php
<?php

declare(strict_types=1);

require __DIR__ . '/../vendor/autoload.php';

use App\Presentation\Http\Request;
use App\Presentation\Http\Response;
use App\Presentation\Routing\Router;
use App\Presentation\Routing\NotFoundException;
use App\Presentation\Routing\MethodNotAllowedException;

session_start();   // CSRF / レートリミットでセッションを使うため

$request = Request::fromGlobals();

$router = new Router();   // php-di を使うなら: new Router(fn($c) => $container->get($c))
require __DIR__ . '/../src/Presentation/Routing/routes.php';

try {
    $response = $router->dispatch($request);
} catch (NotFoundException) {
    $response = Response::error(404, 'Not Found');
} catch (MethodNotAllowedException $e) {
    $response = Response::error(405, 'Method Not Allowed')
        ->withHeader('Allow', implode(', ', $e->allowed));
} catch (\Throwable $e) {
    // 開発時は詳細、本番は汎用（config/app.php の env で切替）
    $response = Response::error(500, 'Internal Server Error');
}

$response->send();
```

### 処理の流れ（再掲）

```
ブラウザ
  → public/index.php
  → Request::fromGlobals()   ($_SERVER/$_GET/$_POST を Request に詰める)
  → routes.php でルート登録
  → Router::dispatch($request)
       ├ resolve(): 静的→動的の順でマッチ。404/405 を例外で表現
       ├ パスパラメータを Request に注入
       ├ ミドルウェア パイプライン実行
       └ コントローラ実行 → Response
  → Response::send()  (http_response_code + header + echo)
  → ブラウザ
```
